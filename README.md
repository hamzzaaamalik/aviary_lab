## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/proto-project.git
   cd proto-project
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the application:
   ```bash
   npm start
   ```
