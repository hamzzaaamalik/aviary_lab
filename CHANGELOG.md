## [0.1.0]

### Added
- `EventBus` — dependency-free typed pub/sub with isolated handlers.
- `ModuleRegistry` — dependency-ordered module boot with cycle detection.
- `Proto` — faculty composition with a perceive → think → act cycle that degrades gracefully.
- `createKernel()` — the shared runtime context.
- Public API surface exported from `src/index.js`.
