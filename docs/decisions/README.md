# Architecture decisions

Every significant, hard-to-reverse choice in this codebase is recorded here as an
[Architecture Decision Record](https://adr.github.io/) — one short file, numbered, immutable once
merged. If a later decision changes an earlier one, write a NEW record that supersedes it rather
than editing history.

Write one when a change decides something a future maintainer would otherwise have to guess at:
a public API shape, a dependency policy, a boundary between modules. Do not write one for routine
work.

Copy [`0000-template.md`](0000-template.md) and take the next free number.
